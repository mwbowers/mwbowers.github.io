#### Syntax 
<pre class="prettyprint"> **BegSr Detach Access(*Public) Type(Void)
   DclSrParm *shareFileName*  Type(*String)**       </pre>  

#### Parameters
<dl>
        <dt>
          <code> *shareFileName* </code>
        </dt>
        <dd>The shared file name to be detached from the 
        <code> **WebDevice** </code>.</dd>
</dl>
